# Sentiment Analyzer Models

This folder will contain trained sentiment analysis models after you run `../notebooks/02_sentiment_analysis.ipynb`.

## Currently Empty?

**This is normal!** Run the notebook to generate models.

## After Training

You'll see:
- `model.pkl` - Trained Naive Bayes model
- `vectorizer.pkl` - TF-IDF vectorizer
- `metadata.json` - Model accuracy
- `nb_confusion_matrix.png` - Evaluation chart
