# Unified AI Business Assistant

Complete standalone desktop application with AI-powered features.

## Quick Start

See [COMPLETE_GUIDE.md](./COMPLETE_GUIDE.md) for full documentation.

## Building

```bash
cd frontend
npm install
npm run build:win
```

Output: `frontend/build/Unified AI Assistant Setup.exe`

## Features

- ✅ Standalone desktop application
- ✅ AI-powered autofill
- ✅ Flexible CMS system
- ✅ Business management (Companies, Customers, Invoices, Payments)
- ✅ Auto-installer for dependencies
